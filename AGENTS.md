# Tessera Project Agent Rules

Tessera is a collaborative photo journal.

Key concepts:

- A **Mosaic** is a shared blog between a group of users.
- A **Tessera** is a single photo entry within a mosaic.

Guidelines for code changes:

1. Prefer simple, readable code.
2. Avoid heavy dependencies.
3. Maintain modular architecture.
4. Update documentation when data structures change.
5. Respect privacy-first design. 

Use Tailwind CSS for styling. Do not introduce React Bootstrap, Material UI, or other heavy component frameworks unless explicitly requested.
