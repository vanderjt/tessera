# Tessera Application Flow

This document describes how users move through the system.

## Primary User Journey

1. User creates an account
2. User joins or creates a mosaic
3. User views mosaic timeline
4. User uploads a tessera (entry)
5. Friends contribute additional entries

Over time the mosaic becomes a shared life archive.

---

## Main Pages

### Home

Landing page introducing Tessera.

Functions:

- explain the concept
- login / sign up
- preview example mosaics

---

### My Mosaics

Dashboard listing mosaics the user belongs to.

Functions:

- view mosaics
- create mosaic
- join mosaic via invite

---

### Mosaic Page

Displays entries for a specific mosaic.

Functions:

- browse entries
- upload new entry
- view contributors

---

### Entry Page

Displays a single entry.

Functions:

- view photos
- read caption
- see author

---

### Create Entry

Upload a new tessera.

Fields:

- photos
- caption
- optional title

---

## Permissions

Only mosaic members may:

- view private mosaics
- post entries
- see contributors

Owners may:

- invite users
- manage roles
- modify mosaic settings
