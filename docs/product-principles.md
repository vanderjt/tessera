# Tessera Product Principles

Tessera is a collaborative photo journal built around small shared moments.

Each photo is a **tessera** — a tile.
Together they form a **mosaic** — a shared story.

The product should emphasize intimacy, memory, and quiet collaboration rather than social media dynamics.

## Core Principles

### 1. Small Moments Matter
Tessera is about everyday life, not highlight reels.

Posts should feel personal and authentic.

### 2. Private by Default
Mosaics exist primarily for groups of friends or family.

Privacy and trust are fundamental.

### 3. Collaborative Storytelling
Multiple people contribute to the same mosaic.

No single voice dominates the narrative.

### 4. Calm Design
The interface should feel:

- quiet
- reflective
- archival

Avoid noisy UI patterns common in social media.

### 5. Longevity
Tessera should feel like a **long-term memory archive**, not a fleeting feed.

Posts should remain meaningful years later.

### 6. Simplicity First
Prefer fewer features that work beautifully.

Avoid unnecessary complexity.

## Anti-Goals

Tessera should **not** become:

- a popularity contest
- a follower network
- an algorithmic feed
- a high-volume content stream

The goal is connection, not engagement metrics.
